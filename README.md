# 每日网络服务自动化巡检脚本

## 🚀 项目简介
这是一个基于 Python 开发的自动化运维工具，每天凌晨自动检测常用服务（百度、阿里云、GitHub、学校VPN等）的连通性。

## ✨ 核心功能
- **连通性测试**：利用 ICMP 协议进行 Ping 检测。
- **自动重试机制**：连续 3 次失败才会触发告警，有效避免网络波动造成的误报。
- **数据持久化**：每日延迟记录自动存入 `inspection_log.csv`。
- **可视化报告**：每日更新 `daily_report.png` 柱状图。
- 支持 HTTP 协议深度检测，监控业务可用性及响应状态码
- V2.0 更新： 引入 HTTP 状态码监控。能够识别 Ping 无法发现的业务层故障（如 503 服务不可用、404 页面丢失），并记录完整的业务响应时间。
- 集成飞书 Webhook，实现毫秒级故障主动告警
